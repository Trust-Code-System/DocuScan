api Endpoint: /v1/scan

Submit a document image for processing. Returns extracted text, structural analysis, and AI-derived metadata ✨.

POST https://api.docuscan.app/v1/scan

Parameters

  • file (required): The document image (JPG, PNG, PDF).
  • extract_tables (optional): Boolean to enable table parsing.
curl -X POST https://api.docuscan.app/v1/scan \
  -H "Authorization: Bearer sk_live_..." \
  -H "Content-Type: multipart/form-data" \
  -F "file=@invoice_october.pdf" \
  -F "extract_tables=true"
Response 200 OK
